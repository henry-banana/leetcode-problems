## 🔄 Quy trình giải

1. **Đọc đề** (5 phút) — xác định input/output, constraints
2. **Nghĩ approach** (10 phút) — brute force trước, optimize sau
3. **Code** (15-20 phút) — clean code, meaningful variable names
4. **Test** — chạy examples + edge cases
5. **Analyze** — Time & Space complexity
6. **Stuck > 25 phút?** → xem NeetCode video, hiểu rồi tự code lại
