## Quy trình giải mỗi bài

1. Đọc đề, nghĩ 15-20 phút → vẽ approach trên giấy/comment
2. Code solution, test locally
3. Submit trên LeetCode
4. Nếu stuck > 25 phút → xem NeetCode video giải, ghi note pattern
5. Redo bài stuck sau 3-7 ngày

- Format: `P{number}_{PascalCaseName}.java`
- Ví dụ: `P0001_TwoSum.java`, `P0217_ContainsDuplicate.java`
- Mỗi file có comment block đầu:

